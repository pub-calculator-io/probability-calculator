The probability calculator can find two events' probability and the normal distribution probability. Learn more about probability's laws and calculations.

[https://www.calculator.io/probability-calculator/](https://www.calculator.io/probability-calculator/)

== Usage ==

Add the Probability Calculator shortcode to your page, post or sidebar:

`[ci_probability_calculator]`

Add the following code to your template where you would like the Probability Calculator to appear:

`<?php display_ci_probability_calculator(); ?>`

== Installation ==

1. Upload the Probability Calculator /ci_probability_calculator/ folder to the /wp-content/plugins/ directory.
2. Activate the Probability Calculator plugin through the "Plugins" menu in WordPress.
